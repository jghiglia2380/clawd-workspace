# Context Folder

This folder contains documents, exports, and reference materials that help Thomas understand your business.

## Suggested Structure

- `emails/` - Important email threads (exported or copied)
- `docs/` - Strategy docs, plans, specs
- `exports/` - Data exports from various systems
- `competitors/` - Competitor research and analysis

## How to Add Context

Drop files here and tell Thomas about them. He'll read and incorporate them into his understanding.
